¶ SYNC_ALL

Player waits that all the clients have terminated the current step before going to the next

 
    exports.stepRules.SYNC_ALL = function(stage, myStageLevel, pl, game) {
        return myStageLevel === node.constants.stageLevels.DONE &&
            pl.isStepDone(stage);
    };

¶ SOLO

Player proceeds to the next step as soon as the current one is DONE, regardless to the situation of other players

 
    exports.stepRules.SOLO = function(stage, myStageLevel, pl, game) {
        return myStageLevel === node.constants.stageLevels.DONE;
    };
