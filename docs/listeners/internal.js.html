Copyright(c) 2016 Stefano Balietti MIT Licensed

Listeners for internal messages.

Internal listeners are not directly associated to messages, but they are usually responding to internal nodeGame events, such as progressing in the loading chain, or finishing a game stage.

http://nodegame.org

        this.events.ng.on(CMD + gcommands.start, function(options) {
            if (!node.game.isStartable()) {
                node.err('"' + CMD + gcommands.start + '": game cannot ' +
                         'be started now.');
                return;
            }
            node.emit('BEFORE_GAMECOMMAND', gcommands.start, options);
            node.game.start(options);
        });
